# 🔐 Password Strength Checker

A simple and fast web app that checks how strong your password is in real time. It helps users create secure passwords to protect their accounts and data.

## 🚀 Live Demo

👉 [Live Demo](https://password-strength-checker-wheat.vercel.app/)

## ✨ Features

- 🔍 Real-time password strength checking
- 🔐 Detects weak, medium, and strong passwords
- ⚡ Instant feedback while typing
- 🎯 Simple and user-friendly interface
- 📱 Responsive design (works on all devices)

## 🛠️ Tech Stack

- 🌐 HTML
- 🎨 CSS
- ⚙️ JavaScript

## 📂 How It Works

1.  User enters a password
2.  App checks length and complexity
3.  It analyzes use of:
    - Uppercase letters
    - Lowercase letters
    - Numbers
    - Special characters
4.  Then shows strength level instantly
